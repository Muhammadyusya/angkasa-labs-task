# Angkasa Labs Task Manager
Aplikasi prototipe antarmuka statis untuk manajemen tugas Angkasa Labs.

Fitur Aplikasi
1. Input: Form penambahan tugas baru.
2. Proses: Simulasi update status tugas.
3. Output: Tabel daftar tugas.

Dependensi
- Web Browser (Chrome/Firefox/Edge).
- Tidak memerlukan database atau server lokal (XAMPP/Laragon).

Cara Menjalankan Aplikasi
1. Clone repositori ini.
2. Buka folder `src/`.
3. Double-click file `index.html` untuk membuka aplikasi di browser.
